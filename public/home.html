About The Project

The Bright Ideas Bird Table is a project for the Biomedical Problem Solving module at the University of Reading. We were tasked to create a bird table, according to our own specifications.

We decided to create a bird table which captured images and videos of birds using a raspberry pi and PIR motion detector. The images are sent to a Convolutional Neural Network (CNN) hosted on a VPS webserver, via a http POST request. The api processes the request, passes the image to the classifier, and then sends the results back to the raspberry pi. The bird table will then play the bird's mating call according to its species, in order to observe its behaviour.


Classifier

The classifier was developed in python using the Keras framework. It was built by transfer-learning the Inception-V3 CNN, with weights from iNaturalist 2017. The CNN can classify 71 different bird species found on campus, using local bird species data gathered by Thomas Whitlock in the Whiteknights Biodiversity project. The dataset was gathered through a web scraper that collected around 150 images per class, with a total of 10000+ images in the whole dataset.


Website and API

The website frontend is built from scratch using HTML, CSS and JavaScript. The API and website backend was built in TypeScript using the NodeJS framework. As we are dealing with images, we had to find a way to send images through a request body, which requires a string format. The solution we found was to convert the image into a base64 string, and send that to the API. The API then sends the base64 encoded image to the classifier, which returns the species of the bird. The bird species information is then sent to the client in a JSON response. All the bird images are automatically saved and uploaded to the website under Bird Captures.
